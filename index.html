Zmart Dashboard Sync

This page hosts the static synchronization script for the Zmart Trading Bot Dashboard.

How to Use

Add the following script tag to your dashboard to enable synchronization with Airtable:

<script src="https://zmart-dashboard.vercel.app/static/dashboard-sync.js"></script>

Features

  • Automatically syncs dashboard prices with Airtable data
  • Adds a "Sync Prices" button to the dashboard
  • Intercepts existing update buttons to ensure data consistency
  • Provides visual feedback for price updates
  • Sets up automatic updates every 5 minutes
